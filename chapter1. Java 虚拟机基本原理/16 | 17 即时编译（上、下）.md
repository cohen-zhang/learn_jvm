## 分层编译模式

- 我使用的 Java 8 默认开启了分层编译，分为 5 层（0层为解释执行）
- 专业术语： profiling（侧面），记录程序运行状态数据； 比如 JDK 的 hprof； 
- 判断是否为热点代码：1。 方法调用计数器 2。循环回边计（字节码简单理解为往回跳转指令）数器
- OSR 编译：基准测试中用，不过多了解
